# Smart Attendance Management System

A web-based attendance system that automates student attendance using location and time verification. This system eliminates manual processes and reduces proxy attendance by ensuring students are physically present during lectures.

##  Features
-  Location-based attendance verification  
-  Time-restricted attendance marking  
-  Dashboard with attendance statistics  
-  Calendar view of lectures  
-  User authentication system  

##  Tech Stack
- **Frontend:** React.js, Tailwind CSS, Vite  
- **Backend:** Java (Spring Boot) *(in progress)*  
- **Database:** MySQL  
- **Other:** Geolocation API  

##  How it Works
1. User logs into the system  
2. Dashboard displays current lecture details  
3. System verifies user location and time  
4. Attendance is marked if conditions are valid  
5. Data is stored and displayed in dashboard  

##  Goal
To build a smart, efficient, and secure attendance system that improves accuracy and reduces manual effort.

##  Future Scope
- Face recognition-based attendance  
- Mobile application  
- QR code-based backup system  
- AI-based attendance analytics
